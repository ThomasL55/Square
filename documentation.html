How to Play

Game Rules

  1. Watch carefully as the game displays a sequence of colored squares
  2. After the sequence ends, repeat the pattern by clicking/tapping the squares in the same order
  3. Each correct sequence adds to your score
  4. If you make a mistake, the game ends and shows your final score
  5. Click "Start Game" to begin a new game

Controls

  • Mouse/Touch: Click or tap directly on the squares
    • {% if settings.use_keyboard %}
  • Keyboard:
    • Arrow keys to move the selection
    • Space bar to select a square
    • ESC to reset current attempt
    • {% endif %}

Accessibility Features

  • High Contrast Mode: Enhances visibility with stronger color contrast
  • Keyboard Controls: Full game control without using a mouse
  • Sound Effects: Audio feedback for game actions
  • Pattern Speed: Adjustable timing for pattern display
